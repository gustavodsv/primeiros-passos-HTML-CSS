Tags de <input>

O HTML 5 veio para facilitar a vida do desenvolvedor web, o que precisava de muito trabalho inclusive de JavaScript,
hoje em um simples código é possivel conseguir o que se quer.

Exemplos:

input de pesquisa

input de email

input de url

input de telefone

quando clica pelo celular, abre o teclado numérico

input de numeros

type="number"

input de slider

slider, type="ranger" /*min = 0, max = 100*/

input de data

type="date"

input de mês

type="month"

input de semana

type="week"

input de tempo

type="time"

input de data e hora

type="datetime-local"

input de cor

type="color"

input de botão(submit)

type="submit"
