Atributo "data-(...)"

Com o atributo: data, é possivel salvar informações no HTML 5,
para poder usar junto com javascript e outras linguagens de interação.

curiosidades

Esse atributo é muito usado no Facebook, para salvar informações do usuário.
Várias bibliotecas do JavaScript é baseada no uso dessa variável.

exemplo

<div data-nome: Gustavo; data-sobrenome: Venancio; data-tipo-pagina:exercicios;>
