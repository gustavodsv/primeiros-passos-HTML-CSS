Tag para Vídeos

Uma das tag que mais chamou atenção no lançamento do HTML5
foi a tag <video>.

        Atributos comuns usados na tag <video>:
        1. width="";
        2. height="";
        3. controls = adiciona os botões e as informações do video.
        4. autoplay = quando  a pagina é carregada, o audio inicia automaticamente.
    
        Na tag source é possivel especificar 3 types:
        1.video/mp4
        2.video/ogg
        3.video/webm
    
    <video width="220" height="140" controls autoplay>
        <source src="link/diretório" type="um dos 3 tipos">
    </video>
