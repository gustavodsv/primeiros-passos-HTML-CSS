Tag para Áudios

Da mesma maneira que é aplicado o video, o audio funciona

Atributos comuns usadas na tag <audio>:
        1. controls = adiciona os botões e as informações do video.
        2. autoplay = quando  a pagina é carregada, o audio inicia automaticamente.
        3. loop = quando o video acaba, ele reinicia.
        4. muted = video inicia mutado.
        5. preload = em casa de internet lenta, o loader da uma pre carregada.

O atributo type aplicado no <source> também suporta arquivos: .mp3, .ogg, .wav.

        <audio controls autoplay loop muted preload>
            <source src="som.mp3" type="audio/mpeg"> 
        </audio>
