Media Query

O @media trabalha com responsividade.

Onde é possivel alterar configurações de exibição da página a partir do tamanho da tela.

        @media only screen and (max-width: 600px){
            .teste {
                background-color: #00ff00;
                width: 300px;
            }
        }
    
        Atributos comuns: 
        max-width = maximo de largura
        min-width = minimo de largura
        orientation:landscape = usado para aparelhos virados na horizontal
