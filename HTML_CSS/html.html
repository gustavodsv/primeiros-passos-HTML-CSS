O que é HTML?

   
    HTML é uma linguagem de marcação utilizada na construção de páginas na Web. 
    Documentos HTML podem ser interpretados por navegadores.

Estrutura do HTML

    <html >
        <head>
            <title>Título da página</title>
        </head>

        <body>
            [conteudo...]
        </body>
    </html>

Cabeçalho

        <head>
            <title>Título da página</title>

            <link/>

            <script /script>

            <style /style>

            <meta/>

            <base href="">

        <head>

Parágrafo

Usa-se <p>[texto]</p> para informar um parágrafo

<pre>[texto formatado(não recebe formatação)]</pre>


exemplo: com <br> usando <p> 
    Lorem ipsum dolor sit amet consectetur adipisicing <br>
